# Code of Conduct

This project expects direct, respectful, and constructive collaboration.

## Expected Behavior

- Be specific when reporting bugs.
- Critique code and behavior, not people.
- Keep discussions focused on the project.
- Respect privacy when sharing calendar-related examples.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Publishing private calendar data or meeting links
- Spam or unrelated promotion
- Deliberately misleading bug reports or security claims

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this code of conduct.
